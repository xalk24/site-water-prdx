# Вода в такси

Самодостаточный адаптивный статический лендинг фиджитал-рекламы на бутылках воды в такси.

Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

Подробная инструкция по публикации и подключению формы: [README_DEPLOY.md](README_DEPLOY.md).
